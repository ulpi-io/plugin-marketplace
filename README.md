# Plugin Marketplace

A curated collection of 4273+ agent skills, packaged as plugins for easy installation in the Claude Desktop app.

All skills are sourced from [skills.sh](https://skills.sh) — the open agent skills ecosystem. This marketplace re-packages them so Claude Desktop users can install them with a single command.

## Installation (Claude Desktop)

1. Open Claude Desktop
2. Go to **Cowork** → **Customize** → **Skills**
3. Above "Personal plugins", click the **+** button
4. Select **Browse** → **Personal**
5. Click **+** again
6. Choose **Add marketplace from GitHub**
7. Enter: `ulpi-io/plugin-marketplace`

Once added, you can browse and install any of the plugins below.

## Installation (CLI)

```bash
claude plugin marketplace add ulpi-io/plugin-marketplace
```

Then install any plugin:

```bash
claude plugin install frontend-design@ulpi-marketplace
```
