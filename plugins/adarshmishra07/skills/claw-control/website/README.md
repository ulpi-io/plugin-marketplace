# Claw Control Website

Marketing/landing page for Claw Control. This is separate from the dashboard that users deploy.

## Development

```bash
cd website
npm install
npm run dev
```

## Deployment

Build and deploy to any static host (Vercel, Netlify, GitHub Pages):

```bash
npm run build
# Output in dist/
```

## Note

This website is for marketing claw-control.dev (or wherever you host it).
The actual dashboard that users deploy is in `/packages/frontend`.
