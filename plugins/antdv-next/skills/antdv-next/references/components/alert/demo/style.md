# More types

## Description (en-US)

There are 4 types of Alert: `success`, `info`, `warning`, `error`.

## Source

```vue
<template>
  <a-alert title="Success Text" type="success" />
  <br>
  <a-alert title="Info Text" type="info" />
  <br>
  <a-alert title="Warning Text" type="warning" />
  <br>
  <a-alert title="Error Text" type="error" />
</template>
```
