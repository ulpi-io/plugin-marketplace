# Custom Render Type

## Description (en-US)

Customize the rendering results by `type`, provide options `canvas` and `svg`.

## Source

```vue
<template>
  <a-space>
    <a-qrcode type="canvas" value="https://www.antdv-next.com" />
    <a-qrcode type="svg" value="https://www.antdv-next.com" />
  </a-space>
</template>
```
