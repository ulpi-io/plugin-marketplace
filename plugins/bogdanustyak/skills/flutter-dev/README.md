# Flutter Development Skill

Expert Flutter and Dart development guidance agents.

## What This Is

A comprehensive skill that provides Flutter best practices, patterns, and modern development standards for building beautiful, performant applications across mobile, web, and desktop platforms.

## Features

- Modern Flutter & Dart best practices
- State management patterns (ValueNotifier, ChangeNotifier, Streams)
- Navigation with GoRouter
- Material Design 3 theming
- Performance optimization techniques
- Testing strategies
- Accessibility guidelines
