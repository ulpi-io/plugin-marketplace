# Cocos2d-x 引擎概述

## 官方文档

参考官方文档：https://docs.cocos.com/cocos2d-x/v4/manual/zh/

## 概述

Cocos2d-x 是一个开源的跨平台游戏开发框架，支持 C++ 开发，可以发布到多个平台。

## 引擎特性

### 跨平台支持
- **桌面平台**: Windows, macOS, Linux
- **移动平台**: Android, iOS
- **Web平台**: 通过 Cocos Creator 支持

### 核心功能
- 2D 图形渲染
- 精灵（Sprite）系统
- 动作（Action）系统
- 场景（Scene）管理
- 物理引擎集成（Box2D）
- 音频系统
- 粒子系统
- 动画系统

### 技术特点
- 基于 OpenGL ES 渲染
- 使用 C++ 开发
- CMake 构建系统
- 高性能渲染管线
- 支持自定义 Shader

## 版本信息

- **当前版本**: Cocos2d-x v4
- **主要语言**: C++
- **构建系统**: CMake
- **渲染后端**: OpenGL ES

## 适用场景

- 2D 游戏开发
- 跨平台游戏应用
- 教育类应用
- 工具类应用
