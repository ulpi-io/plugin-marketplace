# Validation checklist (no-hallucination)

## Structure
- [ ] Report follows `assets/report-template.md` headings
- [ ] Every section includes: 结论要点 / 详细情况 / 证据与出处
- [ ] Final section includes 来源清单（SEC filings + other sources, reverse-chronological)

## Evidence
- [ ] Every factual claim has a citation string
- [ ] Missing facts are expressed as `【...】` placeholders
- [ ] SEC citations include Form + 年度/日期 + 章节/标题
- [ ] Web citations include 机构/网站 + 发布日期 + 标题
- [ ] Real-time data includes an “as-of” timestamp and a user verification note

## Safety
- [ ] No investment advice or price targets
- [ ] No claims without sources
