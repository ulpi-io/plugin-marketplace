# spline-3d-integration

Agent skill for integrating Spline 3D scenes into React, Next.js, and vanilla web apps.

## Install

```bash
npx skills add https://github.com/Fawaz-I/spline-3d-integration -g -y
```

## Verify

```bash
npx skills list
```
