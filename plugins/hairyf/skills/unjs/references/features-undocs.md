# Undocs

Undocs is a minimal documentation theme and CLI tool designed for shared usage across UnJS projects, providing a consistent documentation experience.

## Usage

### CLI Usage

```bash
undocs dev
undocs build
```

### Configuration

```typescript
// undocs.config.ts
export default {
  title: 'My Project',
  description: 'Project documentation',
  theme: 'default',
}
```

## Key Points

- Minimal: Clean and simple documentation theme
- Consistent: Shared theme across UnJS projects
- CLI: Command-line tool for building docs
- Type-safe: Full TypeScript support
