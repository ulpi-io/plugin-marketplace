# Data Source / 数据来源声明

## Music Genre Data

The genre data in this directory is sourced from **[RateYourMusic](https://rateyourmusic.com/genres/)**.

- **Total genres**: 5,947
- **Data date**: 2026-01-31
- **Structure**: 49 main genres > 737 sub-genres > 5,161 detailed sub-genres
