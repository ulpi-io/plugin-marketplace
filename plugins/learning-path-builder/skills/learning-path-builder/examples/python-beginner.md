# Python Learning Path - Beginner to Intermediate

**Duration**: 8 weeks
**Time Commitment**: 8-10 hours/week
**Target Level**: Comfortable building Python applications

## Assessment

- **Current Level**: Programming beginner
- **Learning Style**: Hands-on with projects
- **Goals**: Build web applications with Python
- **Prior Experience**: Basic computer skills

## Week 1-2: Python Fundamentals

**Concepts**: Variables, data types, control flow, functions

**Resources**:

1. [Python.org Tutorial](https://docs.python.org/3/tutorial/) - Chapters 1-4 (4 hours)
2. [Exercism Python Track](https://exercism.org/tracks/python) - First 10 exercises (4 hours)
3. **Project**: Build a calculator CLI app (2 hours)

**Checkpoint**: Can write basic Python programs with functions

## Week 3-4: Data Structures & File I/O

**Concepts**: Lists, dictionaries, file operations, error handling

**Resources**:

1. Python Tutorial - Chapters 5-7 (3 hours)
2. [Real Python - Working with Files](https://realpython.com/working-with-files-in-python/) (2 hours)
3. **Project**: Build a todo list app that saves to file (5 hours)

**Checkpoint**: Can work with data structures and persist data

## Week 5-6: Object-Oriented Programming

**Concepts**: Classes, inheritance, modules, packages

**Resources**:

1. Python Tutorial - Chapters 9-10 (3 hours)
2. [Real Python - OOP](https://realpython.com/python3-object-oriented-programming/) (3 hours)
3. **Project**: Refactor todo app with OOP design (4 hours)

**Checkpoint**: Can design and implement classes

## Week 7-8: Web Development Basics

**Concepts**: HTTP, Flask, templates, forms

**Resources**:

1. [Flask Quickstart](https://flask.palletsprojects.com/quickstart/) (2 hours)
2. [Flask Tutorial](https://flask.palletsprojects.com/tutorial/) (4 hours)
3. **Project**: Build web version of todo app with Flask (6 hours)

**Checkpoint**: Can build simple web applications
