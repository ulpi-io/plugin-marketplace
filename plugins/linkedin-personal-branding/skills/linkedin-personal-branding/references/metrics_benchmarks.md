## Social Selling Index (SSI)

The SSI is LinkedIn's proprietary score (0-100) measuring social selling effectiveness over the last 90 days.

### SSI Components (25 points each)

| Pillar | Description | How to Improve |
|--------|-------------|----------------|
| Establish Professional Brand | Profile completeness, content sharing, follower engagement | Complete profile, share valuable content, grow followers |
| Find the Right People | Using search effectively, identifying prospects | Use advanced search, save leads, view relevant profiles |
| Engage with Insights | Sharing/engaging with relevant content | Comment thoughtfully, share industry news, publish content |
| Build Relationships | Connecting and nurturing relationships | Send personalized invites, respond to messages, engage regularly |

### SSI Score Benchmarks

| Score Range | Level | Percentile |
|-------------|-------|------------|
| 80-100 | Elite | Top 10% |
| 70-79 | Strong | Top 25% |
| 60-69 | Good | Top 40% |
| 40-59 | Average | Typical LinkedIn user |
| Below 40 | Low | Underutilizing platform |

**Target:** Aim for 70+ SSI score minimum

**How to Check SSI:** Visit linkedin.com/sales/ssi

### SSI Impact Statistics
- Users with high SSI generate 45% more opportunities
- 51% more likely to hit sales targets
- 78% outsell peers who don't use social selling

## Content & Engagement Metrics

### Engagement Rate Formula

```
Engagement Rate = (Reactions + Comments + Shares) / Impressions × 100
```

### Engagement Rate Benchmarks

| Rate | Level | Interpretation |
|------|-------|----------------|
| 8%+ | Excellent | Highly resonant content |
| 5-8% | Very Good | Above average engagement |
| 2-5% | Good | Healthy engagement (B2B target) |
| 1-2% | Average | Room for improvement |
| <1% | Low | Content not resonating |

## Engagement Rate Deep Dive

### Calculation Examples

**Example 1 - Low Engagement:**
```
Post: 1,376 impressions, 15 reactions, 1 comment, 0 shares
Engagement Rate = (15 + 1 + 0) / 1,376 × 100 = 1.16%
Status: 🔴 Below target (aim for 3%+)
```

**Example 2 - Good Engagement:**
```
Post: 2,500 impressions, 75 reactions, 12 comments, 3 shares
Engagement Rate = (75 + 12 + 3) / 2,500 × 100 = 3.6%
Status: ✅ On target
```

**Example 3 - Excellent Engagement:**
```
Post: 5,000 impressions, 300 reactions, 45 comments, 20 shares
Engagement Rate = (300 + 45 + 20) / 5,000 × 100 = 7.3%
Status: 🟢 Excellent - viral potential
```

### Improving Low Engagement

If engagement rate is below 2%:
1. **Hook problem**: First line not compelling enough
2. **Content-audience mismatch**: Topic doesn't resonate
3. **Posting time**: Wrong time for your audience
4. **Format issue**: Try different content format
5. **CTA missing**: Add clear call-to-action

### Engagement Rate by Follower Count

| Follower Count | Expected Engagement Rate |
|----------------|-------------------------|
| Under 1,000 | 4-8% (small but engaged) |
| 1,000-5,000 | 3-5% (growth phase) |
| 5,000-10,000 | 2-4% (scaling) |
| 10,000-50,000 | 1.5-3% (broad reach) |
| 50,000+ | 1-2% (mass audience) |

*Note: Higher follower counts typically have lower engagement rates due to audience dilution.*
