# HTML Generation Specifications

## Basic Specifications

| Item | Specification |
|------|---------------|
| Resolution | 1280 × 720 (required) |
| Chart library | Chart.js 4.5.0 |
| Icon library | Font Awesome 6.5.2 |

### CDN References

```html
<script src="https://cdn.jsdelivr.net/npm/chart.js@4.5.0/dist/chart.umd.js"></script>
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/@fortawesome/fontawesome-free@6.5.2/css/all.min.css">
```

---

## Theme Rules

Strictly follow the theme's colors, fonts, and decoration style. See [theme.md](theme.md) for details.

Avoid using the same theme template for consecutive pages to maintain visual diversity.

---

## Content Rules

### Strictly Follow the Outline

| Rule | Description |
|------|-------------|
| No modifications | Do not alter the outline's text content |
| No expansion | Do not add content not in the outline |
| No abbreviation | Do not remove outline content |
| Must implement | Charts, lists, and images specified in the outline must be implemented |

Emphasized content marked with `**text**` in the outline should be highlighted with visual styles — do not copy the `**` markers directly.

### Content Overflow Handling

All content must display completely within 1280×720. When overflowing, prioritize:
1. Reduce font size (body min 14px, title min 24px)
2. Reduce spacing (decrease padding / gap)
3. Split content across multiple pages (notify the user)

---

## Icon Rules

```html
<i class="fas fa-chart-line"></i>  <!-- Solid -->
<i class="far fa-chart-line"></i>  <!-- Outline -->
<i class="fab fa-github"></i>      <!-- Brand -->
```

- Icons may accompany titles and key points; maximum one per module
- Emoji is prohibited — use Font Awesome instead

---

## Image Rules

Only use images uploaded by the user, fetched/generated by tools, or explicitly provided. Do not use image URLs from AI's built-in knowledge.

- Use `object-fit: cover` consistently
- Be mindful of original dimensions to prevent cropping or distortion

---

## Prohibited Items

| Prohibited Item | Reason |
|-----------------|--------|
| Emoji symbols | Use Font Awesome instead |
| Fabricated image URLs | Only use images from user/tools |
| Multiple cover pages | Only one cover per project |
| Multiple ending pages | Only one ending per project |
| Pages before cover | Cover must be the first page |
| Pages after ending | Ending must be the last page |
