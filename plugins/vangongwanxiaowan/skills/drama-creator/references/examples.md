# 竖屏短剧剧本创作大师 - 详细示例

## 示例1：创作打脸场景剧本

### 输入

```
帮我创作一个打脸场景，主角是隐藏大佬身份的总裁装穷体验生活
```

### 输出

```
【分析：这是一个"放弹簧"场景，需要极致的爽感释放】

1. 高档餐厅 日 内
女主苏雅正在和朋友聚餐，前男友带着新女友走进餐厅。

前男友：哟，这不是苏雅吗？怎么，一个人在这儿吃啊？

男主顾言穿着服务员的制服走过来倒水。

女友：这服务员怎么这样，水都倒不干净。

顾言低头道歉：对不起，对不起。

前男友得寸进尺：就是，叫你们经理出来！

餐厅经理匆匆赶来，看到顾言后，直接鞠躬90度。
经理：顾……顾总，您怎么在这里？

前男友愣住：什么顾总？

经理：这位先生，我们集团董事长顾言先生亲自为您服务。

顾言缓缓站直身体。
顾言：经理，把这两位请出去，列入黑名单，顾氏集团旗下所有产业永不接待。

前男友和女友狼狈地被赶出餐厅。
```

## 示例2：创作悬念钩子

### 输入

```
帮我设计一个悬疑短剧的悬念钩子，第一集结尾要让观众想看第二集
```

### 输出

```
【分析：悬念钩子需要强冲突+信息差+情绪峰值】

1. 废弃工厂 夜 内
主角陈默被绑在椅子上，对面是神秘黑衣人。

黑衣人：你知道你父亲是怎么死的吗？

陈默：……

黑衣人拿出一份文件。
黑衣人：这份文件可以证明一切，但看到这份文件的人，都活不过24小时。

陈默挣扎。
陈默：给我！

黑衣人冷笑。
黑衣人：你想看？好，我给你看。

黑衣人将文件扔到陈默面前。

陈默低头看文件，瞳孔放大。

黑衣人拿起枪，对准陈默。
黑衣人：现在，你可以去见你父亲了。

【就在枪响的一瞬间——】

【画面黑屏】
【枪声响起】
【字幕：未完待续】

【分析：这个悬念钩子包含了多个钩子元素】
1. 父亲死因之谜（信息钩子）
2. 文件内容之谜（内容钩子）
3. 24小时死亡诅咒（危机钩子）
4. 结尾枪响（情绪钩子）
```

## 示例3：优化现有剧本

### 输入

```
请帮我优化以下剧本，让它更符合竖屏短剧的特点：

【原剧本】
1. 咖啡厅 日 内
小李在喝咖啡，同事小张走过来。

小张：听说你要辞职了？

小李：是啊，我想换个环境。

小张：那你要去哪里？

小李：还没想好，可能去创业吧。

小张：加油，我支持你。

小李：谢谢。
```

### 输出

```
【分析问题】
1. 对话平淡，缺少冲突
2. 没有悬念，没有情绪峰值
3. 信息量不足，没有吸引点

【优化方案 - 增加"打脸"元素】

1. 咖啡厅 日 内
小李在喝咖啡，同事小张走过来。

小张：（嘲讽）哟，听说你要辞职了？就你这样还能找到工作？

小李：（淡定）不劳你费心。

小张：别是找不到工作被开除吧？哈哈！

此时，一辆豪车停在咖啡厅门口。

车窗摇下，一位穿着西装的男士对小李招手。
男士：李总，会议马上开始了，请上车。

小张愣住：李……李总？

小李起身，整理衣服。
小李：这位是我们集团的副总，来接我去开董事会。

小李走出咖啡厅，上了豪车。

小张目瞪口呆，嘴里的咖啡喷了出来。

【分析优化点】
1. 增加冲突：同事的嘲讽
2. 增加反转：隐藏大佬身份揭露
3. 增加爽感：打脸同事
4. 视觉化：豪车、西装等元素
5. 节奏紧凑：信息前置，直接切入核心
```

## 示例4：设计故事大纲

### 输入

```
帮我设计一个重生复仇题材的竖屏短剧大纲
```

### 输出

```
【题材分析】
重生复仇是竖屏短剧的热门题材，核心爽点是"预知未来"+"改写命运"。
需要设计极致的仇恨、极致的复仇、极致的打脸。

【故事大纲】

主角：林晚（女，前世被渣男和闺蜜害死）
反派：顾成（渣男前男友）、苏菲（绿茶闺蜜）

第一集：死亡与重生
- 林晚发现顾成和苏菲的背叛
- 林晚被推下楼身亡
- 林晚重生回到三年前（刚和顾成在一起）

第二集：改变命运
- 林晚决定改变命运
- 故意接近顾成的死对头——霸道总裁傅司寒
- 傅司寒对林晚产生兴趣

第三集：布局复仇
- 林晚开始收集顾成和苏菲的罪证
- 假装还是顾成的女友，实则设局
- 傅司寒成为林晚的靠山

第四集：真相揭露
- 林晚在公开场合揭露顾成和苏菲的阴谋
- 傅司寒出现，支持林晚
- 顾成和苏菲身败名裂

第五集：终极打脸
- 林晚和傅司寒在一起
- 顾成和苏菲在街头乞讨（形成强烈对比）
- 林晚和傅司寒的豪车经过，没有停留
- 顾成看到林晚的车，后悔莫及

【情绪曲线】
压-压-放-压-放
第一集：压（死亡）
第二集：压（忍辱负重）
第三集：压（布局）
第四集：放（真相揭露）
第五集：放（终极打脸）

【悬念钩子】
每集结尾设置悬念：
1. 林晚重生，观众好奇她如何改变命运
2. 傅司寒出现，观众好奇他会如何帮助林晚
3. 林晚设局，观众好奇她如何复仇
4. 真相揭露，观众好奇反派下场
5. 终极打脸，观众获得爽感释放
```
