# 竖屏短剧创作模板

本文件提供各类竖屏短剧的创作模板，包括人物设定、情节结构、场景模板等。

## 创作类型模板

### 1. 现代都市爱情 (romance)

**主题**: 现代都市爱情
**关键元素**: 相遇、误会、和解、表白
**基调**: 温馨浪漫

**主角设定**:
- **男主**:
  - 年龄: 25-30岁
  - 职业: 总裁、高管、专业人士
  - 性格: 阳光开朗或霸道总裁
  - 特点: 专一、宠溺、保护欲强

- **女主**:
  - 年龄: 22-28岁
  - 职业: 职场新人、自由职业者
  - 性格: 温柔善良或独立自强
  - 特点: 纯真、善良、坚韧

**情节结构**:
1. **相遇**: 意外或有意安排的初次见面
2. **相知**: 通过事件加深了解
3. **误会**: 外部阻挠或内部误解
4. **分离**: 暂时分开或被迫分离
5. **重逢**: 克服困难重新在一起
6. **表白**: 情感爆发，确立关系

### 2. 轻松幽默喜剧 (comedy)

**主题**: 轻松幽默喜剧
**关键元素**: 误会、巧合、反转、笑点
**基调**: 轻松幽默

**主角设定**:
- **主角**: 普通人，有鲜明的个性特征
- **配角**: 各有特色的搞笑角色
- **反派**: 滑稽可笑，不是真正的威胁

**情节结构**:
1. **铺垫**: 介绍主角的有趣特点
2. **冲突**: 主角陷入搞笑困境
3. **升级**: 问题越来越复杂
4. **反转**: 意想不到的解决方案
5. **结局**: 欢喜收场

### 3. 情感剧情 (drama)

**主题**: 情感剧情
**关键元素**: 冲突、成长、选择、结局
**基调**: 深刻感人

**主角设定**:
- **主角**: 面临人生选择的人物
- **配角**: 影响主角决定的人
- **反派**: 阻碍主角成长的人

**情节结构**:
1. **现状**: 主角的初始状态
2. **激励事件**: 打破现状的事件
3. **探索**: 主角尝试改变
4. **考验**: 面临困难的选择
5. **高潮**: 做出关键决定
6. **结局**: 接受结果并成长

### 4. 悬疑惊悚 (thriller)

**主题**: 悬疑惊悚
**关键元素**: 悬念、线索、反转、真相
**基调**: 紧张刺激

**主角设定**:
- **主角**: 寻求真相的人
- **配角**: 提供线索或阻碍的人
- **反派**: 隐藏真相的人

**情节结构**:
1. **谜题**: 离奇事件发生
2. **调查**: 主角开始调查
3. **线索**: 发现各种线索
4. **危险**: 主角遇到威胁
5. **真相**: 揭露真相
6. **对决**: 与真凶对决

## 场景模板

### 场景1: 身份揭晓场景

**用途**: 主角隐藏身份被揭露时

**模板**:
```
N. 场景地点 时间 内外景

[被羞辱或被轻视的主角]
[反派嚣张]

[关键人物登场]
关键人物：（震惊）你...

主角：（淡定回应）

[身份揭晓]
关键人物：这是[身份]！

[全场震惊反应]
[反派慌乱]

主角：（霸气宣告）
```

**示例**:
```
1. 高档宴会厅 夜 内
服务员林浅正在清理桌面，被一群名媛嘲笑。

名媛A：哟，这不是林浅吗？怎么在这儿打工啊？

林浅：（低头）这是我的工作。

名媛B：听说你被顾家赶出来了？真可怜。

此时，宴会厅大门打开，一位气宇轩昂的男士走进来。
男士：（环视全场）今晚的宴会，我是主人。

名媛们：（震惊）顾……顾总？

顾总径直走到林浅面前，单膝跪地。
顾总：夫人，你该回来了。

全场倒吸一口凉气。
名媛A、B脸色惨白。
```

### 场景2: 重逢场景

**用途**: 旧情人或故人重逢

**模板**:
```
N. 场景地点 时间 内外景

[主角A在做某事]
[主角B出现]

[两人对视]
[时光倒流的感觉]

[尴尬或激动的对话]
[情感涌动]
```

**示例**:
```
1. 咖啡厅 日 内
苏雅正在看文件，有人走过来停在她桌前。

苏雅：（抬头）请问……

顾远：（微笑）好久不见。

苏雅愣住，手中的咖啡杯掉在地上。

顾远：五年了，你一点都没变。

苏雅：（眼眶发红）你……

顾远：我来履行当年的承诺。
```

### 场景3: 危机救援场景

**用途**: 主角遇到危险，被救助

**模板**:
```
N. 危险场景 时间 内外景

[主角陷入危险]
[反派得意]

[救援者出现]
[激烈对抗]

[主角获救]
[情感爆发]
```

**示例**:
```
1. 废弃仓库 夜 内
林浅被绑在椅子上，几个黑衣人围着她。

黑衣人A：劝你签了吧，别自找苦吃。

林浅：我不会签的。

黑衣人B：那就别怪我们不客气。

黑衣人B举起棍子，就要打下去。

突然，仓库大门被撞开。

顾远：（怒吼）敢动我的人！

几名保镖冲进来，与黑衣人打斗。

顾远冲到林浅身边，解开绳子。
顾远：你没事吧？

林浅：（眼泪流下）你怎么来了？

顾远：（轻擦她的眼泪）我不能再失去你一次。
```

## 人物关系模板

### 关系1: 霸道总裁 x 平凡女孩

**设定**:
- **男主**: 霸道总裁，外冷内热，只对女主温柔
- **女主**: 平凡女孩，善良纯真，不卑不亢
- **关系**: 身份悬殊，真爱战胜一切

**情节要点**:
- 初遇：男主对女主的特别关注
- 误解：女主以为男主只是在玩弄感情
- 真相：男主的深情被揭露
- 阻碍：家庭反对、前女友干扰
- 结局：克服困难，幸福在一起

### 关系2: 隐藏大佬 x 伪装平凡

**设定**:
- **男主**: 隐藏身份的大佬（总裁、战神、神医）
- **女主**: 不知道男主真实身份
- **关系**: 身份差距，最终身份揭晓

**情节要点**:
- 相遇：女主遇到"普通"的男主
- 互动：日常相处，女主逐渐爱上男主
- 危机：女主遇到困难，"普通"男主展现能力
- 揭晓：男主真实身份被揭露
- 反转：女主知道真相后的反应
- 结局：接受真实身份，幸福在一起

### 关系3: 青梅竹马 x 第三者

**设定**:
- **男主**: 与女主青梅竹马
- **女主**: 夹在两个男人之间
- **男二**: 新出现的追求者
- **关系**: 三角关系，最终选择

**情节要点**:
- 现状：女主和男一青梅竹马
- 变化：男二出现，对女主好
- 纠结：女主的心开始动摇
- 冲突：两个男人的对决
- 选择：女主做出最终选择
- 结局：选定一人，另一人释怀

## 剧情结构模板

### 结构1: 逆袭复仇 (10集)

**主题**: 从被压制到逆袭复仇

**分集大纲**:
- 第1集：主角被陷害，身败名裂
- 第2集：主角忍辱负重，寻找机会
- 第3集：主角开始布局，收集证据
- 第4集：反派继续嚣张，观众情绪积累
- 第5集：主角开始反击，小试牛刀
- 第6集：反派恐慌，观众期待爆发
- 第7集：主角身份揭晓，震惊全场
- 第8集：全面反击，打脸所有反派
- 第9集：真相大白，反派身败名裂
- 第10集：主角登顶，反派落魄

**情绪曲线**: 压-压-压-放-放

### 结构2: 甜宠爱情 (10集)

**主题**: 霸道总裁宠妻

**分集大纲**:
- 第1集：男女主相遇，男主对女主特别关注
- 第2集：女主遇到困难，男主暗中帮助
- 第3集：两人相处增多，暧昧滋生
- 第4集：女主遭遇危险，男主英雄救美
- 第5集：男主开始宠溺女主，甜蜜升级
- 第6集：外部阻力出现，两人感情受考验
- 第7集：误会加深，两人暂时分开
- 第8集：真相揭露，误会解除
- 第9集：男主展开追求，浪漫升级
- 第10集：女主接受，甜蜜结局

**情绪曲线**: 甜-虐-甜-甜

### 结构3: 悬疑推理 (10集)

**主题**: 破案解谜

**分集大纲**:
- 第1集：离奇案件发生，警方介入
- 第2集：主角发现线索，开始调查
- 第3集：线索错综复杂，陷入迷局
- 第4集：发现重要线索，方向明确
- 第5集：主角遇到危险，悬疑升级
- 第6集：突破性进展，真凶浮现
- 第7集：真凶反击，主角陷入困境
- 第8集：绝地反击，锁定证据
- 第9集：真相大白，真凶伏法
- 第10集：案件总结，正义伸张

**情绪曲线**: 悬-疑-悬-放

## 对话模板

### 台词1: 霸道总裁式

**特点**: 简洁、有力、不容置疑

```
男主：从今天起，她就是我的女人。
男主：谁敢动她一根头发，我让他全家陪葬。
男主：在这个世界上，只有我能欺负她。
```

### 台词2: 宠溺温柔式

**特点**: 温柔、体贴、保护欲强

```
男主：别怕，有我在。
男主：我会保护你，不让你受一点委屈。
男主：你的愿望，就是我的命令。
```

### 台词3: 反派嚣张式

**特点**: 狂妄、轻蔑、目中无人

```
反派：你也配跟我争？
反派：你以为你有资格吗？
反派：给我跪下！
```

### 台词4: 身份揭晓式

**特点**: 震撼、反转、信息密集

```
配角：他是...
男主：没错，我就是[身份]。
全场：（震惊）
反派：（慌乱）
```

## 创作技巧模板

### 技巧1: 信息前置写法

**错误示范**:
```
A: 你最近怎么样？
B: 还行吧，你呢？
A: 我听说了，要辞职了。
B: 哦，是吗？
```

**正确示范**:
```
A: 我要辞职了，准备创业。
B: 这么突然？有什么计划吗？
```

### 技巧2: 动作可视化写法

**错误示范**:
```
男主非常生气，眼神凶狠地看着反派。
```

**正确示范**:
```
男主一把揪住反派的衣领，将他狠狠撞在墙上。
男主：（咬牙切齿）你敢再说一遍？
```

### 技巧3: 悬念钩子写法

**结尾悬念类型**:
- 身份悬念：他是谁？
- 真相悬念：发生了什么？
- 危机悬念：能否逃脱？
- 反转悬念：接下来会怎样？

**示例**:
```
【悬念示例1 - 身份】
神秘人摘下面具，露出的脸让所有人都震惊了。
【悬念示例2 - 真相】
文件打开的那一刻，所有人都不敢相信自己的眼睛。
【悬念示例3 - 危机】
炸弹倒计时：3...2...1...
【悬念示例4 - 反转】
就在这时，一个意想不到的人出现了。
```
